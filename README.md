## Project Overview

The Stock Portfolio Tracker is a simple Python-based application that helps users calculate the total value of their stock investments. The program allows users to enter stock names and quantities, calculates the investment value based on predefined stock prices, and displays the total portfolio value.

Additionally, the application saves the investment summary to a text file, making it easy for users to keep a record of their portfolio. This project demonstrates the use of Python dictionaries, loops, conditional statements, user input handling, and file operations.

---

## Objective

The objective of this project is to:

- Develop a simple stock portfolio management application.
- Calculate the total investment value based on stock prices.
- Learn the use of dictionaries in Python.
- Understand loops, conditional statements, and file handling.
- Store portfolio details in a text file for future reference.

---

## Features

### Stock Selection

Allows users to select available stocks such as AAPL, TSLA, GOOG, MSFT, and AMZN.

### Quantity Input

Users can enter the quantity of shares they own.

### Investment Calculation

Calculates the value of each stock using the predefined stock price.

### Portfolio Summary

Displays the total investment value after all stocks are entered.

### File Storage

Saves the portfolio summary in a **portfolio.txt** file.

### User-Friendly Interface

Simple command-line interface that is easy to understand and use.

## Program Workflow

1. Display available stocks.
2. Accept stock name from the user.
3. Verify whether the stock exists.
4. Accept the quantity of shares.
5. Calculate the stock investment value.
6. Add the value to the total investment.
7. Repeat until the user chooses to stop.
8. Display the investment summary.
9. Save the total investment value to **portfolio.txt**.
